CalendarDemo
===============

+ Web demo: https://calendar-demo-expo.herokuapp.com/

+ Expo: https://expo.io/@gaboelnuevo/CalendarDemo

+ Install global dependencies

```
npm install -g expo-cli yarn
```

+ Run app:

```
git clone https://github.com/gaboelnuevo/CalendarDemo
```

```
cd DemoCalendar
```

```
yarn
```

```
yarn web
```
